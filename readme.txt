# 💘 Kalkulator Keserasian Nama

Kalkulator cinta interaktif berasaskan **nama anda vs nama si dia**. Masukkan dua nama, sistem akan kira % keserasian secara fun sahaja 😉

👉 Live demo: [GitHub Pages](https://username.github.io/kalkulator-keserasian-nama/)

## ✨ Ciri-ciri
- Input dua nama (anda & crush)
- Kiraan % keserasian (algoritma hash deterministik – result sama untuk nama yang sama)
- Simetri: Ali + Siti = Siti + Ali
- Papar kategori result:
  - 💞 Soulmate Vibes (85% ke atas)
  - 💕 Sweet Match (60–84%)
  - 🙂 Boleh Diusahakan (40–59%)
  - 🤝 Kawan Dulu (bawah 40%)
- Meter animasi dengan warna pinky 💖
- Boleh **share** result terus (Web Share API / WhatsApp / Copy Link)
- URL shareable: contoh `?a=Ali&b=Siti&p=72`

## 🚀 Deploy
1. Buat repo baharu di GitHub: `kalkulator-keserasian-nama`
2. Upload fail `index.html`
3. Pergi ke Settings → Pages → Deploy from a branch → pilih `main` → folder `/root`
4. Selesai! Laman boleh diakses di `https://username.github.io/kalkulator-keserasian-nama/`
